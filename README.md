# README

This repository contains the notebooks and resources that will be used during tutorial sessions. The notebooks are designed to be run in a Jupyter environment, especially within Colab for ease of access.

## How to Use

* To get started, you can download the entire repository as a ZIP file from GitHub. This will allow you to access all the notebooks and resources locally on your machine.

<p align="center">
  <img src="https://raw.githubusercontent.com/okkyibrohim/tp-css-tutorial/main/figure/gh-download.png" alt="Download ZIP" width="300"/>
</p>

* Once you have the ZIP file, extract it to a location of your choice.

* Copy the extracted folder to your Google Drive. This will allow you to access the notebooks directly from Colab.

* Navigate to the folder in your Google Drive where you copied the extracted files, and double-click on the notebook you want to run. This will open the notebook in Colab.

## Structure

The [notebooks](https://github.com/okkyibrohim/tp-css-tutorial/tree/main/notebook) directory contains all the Jupyter notebooks that will be used during the tutorial sessions.

The [figure](https://github.com/okkyibrohim/tp-css-tutorial/tree/main/figure) directory contains images that are referenced in the notebooks.

The [dataset](https://github.com/okkyibrohim/tp-css-tutorial/tree/main/dataset) directory contains any datasets that are used in the notebooks.

## Notebooks

The notebooks are organized in a way that each one focuses on a specific topic or tool. Some of the knowledge from previous notebooks may be used in later notebooks, so it's recommended to follow the sequence for a better understanding.
